# オープンキャンパス用のチャットボット

# 環境の起動の仕方

以下のコマンドを打ちましょう

```
$ source env/bin/activate
```

# モジュールがない時

以下のコマンドを入力してみてください

```
pip install -r requirement.txt
```

## テストするには

随時更新予定


- ngrokのインストール

```
brew cask intall ngrok
```
